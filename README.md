# Manish Adhikari — Portfolio

Personal portfolio website for Manish Adhikari, AI/ML graduate and ICT professional based in Adelaide, South Australia.

**Live site:** [manishadhikari.dev](https://manishadhikari.dev) <!-- TODO: update once domain is live -->

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Skills visualisation:** D3 force graph
- **AI features:** Claude API (Anthropic) — chat widget + cover letter generator

## Features

- Typewriter hero, scroll-triggered animations, glassmorphism dark theme
- Interactive skill constellation (D3 force graph)
- AI chat assistant trained on Manish's background
- AI-powered cover letter generator
- Sections: Hero · About · Skills · Experience · Projects · Education · Contact

## Local Development

```bash
npm install
cp .env.local.example .env.local   # add your ANTHROPIC_API_KEY
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
